# Static

Ce répertoire contient les ressources statiques de l'application.

## Fichiers

### style.css
Feuille de style principale de l'application.

- **Rôle**: Définit l'apparence et la mise en page de l'interface utilisateur
- **Responsabilités**:
  - Style de la calculatrice
  - Disposition des boutons
  - Effets visuels et transitions
  - Adaptation responsive

## Dépendances
- Aucune dépendance externe - utilise uniquement du CSS natif

## Notes pour les développeurs
- Les styles sont organisés par composants
- Privilégier l'utilisation de classes plutôt que d'IDs
- Maintenir la compatibilité avec les différents navigateurs
- Respecter les principes de design responsive
