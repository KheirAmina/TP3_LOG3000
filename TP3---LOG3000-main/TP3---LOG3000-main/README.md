## Ojectif du projet
Ce projet consiste à développer une **application web de calculatrice** à l’aide du micro-framework **Flask** en Python.

L’objectif principal est d’appliquer les bonnes pratiques de **gestion de versions**, de **documentation** et de **tests automatisés**, conformément aux exigences du cours **LOG3000**.  
L’application permet d’effectuer les **opérations arithmétiques de base** (addition, soustraction, multiplication, division) via une interface web simple et intuitive.
